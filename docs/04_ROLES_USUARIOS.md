# ROLES DEL SISTEMA

## Administrador

Control total.

Permisos:

* Usuarios.
* Clientes.
* Conductores.
* Vehículos.
* Servicios.
* Configuración.
* Auditoría.

## Operador

Operación diaria.

Permisos:

* Crear servicios.
* Modificar servicios.
* Asignar conductores.
* Asignar vehículos.
* Actualizar estados.

## Conductor

Uso desde aplicación Flutter.

Permisos:

* Ver servicios asignados.
* Actualizar estados.
* Registrar observaciones.

## Cliente Externo

Portal público.

Permisos:

* Consultar servicios.
* Visualizar historial.
* Descargar documentos autorizados.
