# VISIÓN DEL PROYECTO

## Nombre

Portal de Gestión y Trazabilidad de Servicios Logísticos JHT.

## Objetivo

Implementar una plataforma web que permita registrar, administrar y consultar servicios logísticos realizados por JHT Transport Company, proporcionando trazabilidad en tiempo real de los estados operativos y permitiendo a los clientes consultar el avance de sus servicios mediante un portal público.

## Alcance Estratégico

El sistema NO será diseñado únicamente para seguimiento de guías de remisión.

El sistema será una plataforma de gestión de servicios logísticos capaz de soportar distintos tipos de operaciones:

* Distribución.
* Courier.
* Entregas corporativas.
* Mudanzas.
* Recojos.
* Servicios especiales.
* Servicios personalizados.

Cada servicio podrá poseer uno o varios documentos asociados dependiendo del cliente y del tipo de operación.

## Visión a Futuro

La plataforma deberá permitir incorporar posteriormente:

* Aplicación móvil Flutter.
* GPS.
* Geolocalización.
* Evidencias fotográficas.
* Firma digital.
* Integración WhatsApp.
* Dashboard Gerencial.
* Integraciones con sistemas externos.

La arquitectura deberá prepararse desde el inicio para soportar estas futuras ampliaciones sin rediseños estructurales.
