# MVP OBLIGATORIO

Debe incluir:

* Seguridad.
* Roles.
* Clientes.
* Conductores.
* Vehículos.
* Servicios.
* Destinos múltiples.
* Documentos.
* Tracking público.
* Historial.
* Auditoría.

---

# NO MVP

* GPS.
* Geolocalización.
* WhatsApp.
* Firma Digital.
* Evidencias Fotográficas.
* Dashboard BI.
* Integración ERP.
* Integración SUNAT.
